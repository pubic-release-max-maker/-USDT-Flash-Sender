## 🚀 What Is USDT Flash Sender?

USDT Flash Sender (Simulation Tool) is a blockchain transaction
simulator designed to demonstrate how USDT transfers appear across major
networks:

-   TRC20 (Tron Network)\
-   ERC20 (Ethereum Network)\
-   BEP20 (Binance Smart Chain)

This tool helps users understand wallet behavior, transaction display,
and blockchain mechanics without using real cryptocurrency.

## 🧠 How It Works

The tool simulates blockchain transactions by:

-   Generating mock USDT transaction data\
-   Broadcasting to testnet or simulated endpoints\
-   Mimicking wallet notifications and confirmations\
-   Displaying non-spendable transaction hashes
